## A techstack for the complete development 
this is a simple lightweight login with registration for a online course plateform with a simple techstack
### backend 
- django

### Database
- SQlite(for simple beginning)

### Frontend
- Django Templates (HTML + CSS + JS)

### Authentication
- Use Django’s built-in (django.contrib.auth) or a coustem auth.
```scss
Online Course Simulation
Features:
User Registration
User Login / Logout
Dashboard after login
Course list
Enroll in course
```
