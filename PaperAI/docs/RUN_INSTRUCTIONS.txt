FULL INSTRUCTIONS
═════════════════════════════════════════════════════════════════════════════════

1. SETUP VIRTUAL ENVIRONMENT
────────────────────────────

Why? Virtual environments isolate dependencies and avoid system conflicts.

$ cd /Users/ashithrai/Documents/learn/A-Z/PaperAI
$ python3 -m venv venv

This creates a 'venv' directory with isolated Python environment.

Activate it:
$ source venv/bin/activate

You'll see (venv) prefix in your terminal.


2. INSTALL DEPENDENCIES
───────────────────────

$ pip install -r requirements.txt

This installs ~40 packages including:
  • PyQt5 (GUI framework)
  • Groq (LLM API)
  • FAISS (vector database)
  • Sentence Transformers (embeddings)
  • PyMuPDF (PDF processing)
  • And many more...

First installation takes 2-5 minutes.
Subsequent runs are instant (using cache).


3. SETUP GROQ API KEY
──────────────────────

Get your FREE API key:
  1. Go to: https://console.groq.com/keys
  2. Sign up (email + password)
  3. Create new API key
  4. Copy the key (starts with "gsk_")

Then:
$ cp .env.example .env
$ nano .env

Add your key:
  GROQ_API_KEY=gsk_your_key_here

Save: Ctrl+O, Enter, Ctrl+X


4. RUN THE APPLICATION
───────────────────────

Make sure virtual environment is active:
$ source venv/bin/activate

Then choose your interface:

GUI (Recommended for users):
$ python main.py

A window will open. You can:
  • Click "📄 Upload PDF"
  • Select a research paper
  • Ask questions
  • Get instant answers

CLI (Recommended for automation):
$ python cli.py process papers/my_paper.pdf
$ python cli.py ask "What is the main contribution?"
$ python cli.py explain "Attention mechanisms"
$ python cli.py interactive

TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════════

Issue: "python: command not found"
─────────────────────────────────
Use python3 instead:
$ python3 main.py

Or create an alias:
$ alias python=python3


Issue: "ModuleNotFoundError: No module named 'PyQt5'"
──────────────────────────────────────────────────────
Forgot to activate virtual environment!

$ source venv/bin/activate

Check if (venv) appears in prompt.


Issue: "GROQ_API_KEY not found"
───────────────────────────────
Missing or incorrect .env file:

$ cp .env.example .env
$ nano .env
# Add your key and save


Issue: "error: externally-managed-environment"
───────────────────────────────────────────────
System preventing direct pip install.

Solution: Use virtual environment:
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt


Issue: Installation taking very long
──────────────────────────────────────
PyTorch/FAISS can take time. Patient wait. Or:

Option 1: Use lightweight versions
  $ pip install groq PyQt5 sentence-transformers faiss-cpu python-dotenv PyMuPDF

Option 2: Install core, skip optional
  (remove some dependencies from requirements.txt)
