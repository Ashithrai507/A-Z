🏗️ ARCHITECTURE (5-LAYER SYSTEM)
═══════════════════════════════════════════════════════════════════════════════

User Interface Layer
  ├─ GUI (PyQt5) ................ Click buttons, upload files
  └─ CLI (argparse) ............. Powerful command-line tools

PDF Processing Layer
  ├─ Extraction ................. PyMuPDF extracts text
  ├─ Chunking ................... Split into overlapping segments
  └─ Metadata ................... Preserve document information

Embeddings Layer
  ├─ Model ...................... Sentence Transformers
  ├─ Dimension .................. 384-dimensional vectors
  └─ Processing ................. Batch & GPU support

Vector Database Layer
  ├─ Storage .................... FAISS IndexFlatL2
  ├─ Search ..................... Similarity matching
  └─ Persistence ................ Save/load capability

LLM Layer
  ├─ Provider ................... Groq API
  ├─ Model ...................... llama-3.1-70b-versatile
  └─ Operations ................. Q&A, summarization, explanation
