# A-Z

## LEVEL 1 – Foundation (Must Master First)

These build 60% of interview problems.

1. **Two Pointers**  
2. **Sliding Window**  
3. **Hash Maps**  
4. **Stack**  
5. **Modified Binary Search**

## LEVEL 2 – Linear Structures & Patterns

6. **Fast & Slow Pointers**  
7. **Monotonic Stack**  
8. **In-place Reversal of Linked List**  
9. **Merge Intervals**  
10. **Cyclic Sort**

These teach:

- LinkedList mastery
- Pattern recognition
- Optimization thinking

## LEVEL 3 – Trees & Recursion (Very Important)

11. **Tree Breadth First Search**  
12. **Tree Depth First Search**  
13. **Backtracking**  
14. **Subsets**

This is where interview difficulty increases.

You’ll start understanding:

- Recursion deeply
- Tree traversal patterns
- State space exploration

## LEVEL 4 – Graphs & Advanced Patterns

15. **Graphs**  
16. **Island (Matrix traversal)**  
17. **Topological Sort**  
18. **Union Find**

This is high-value for:

- Google
- Amazon
- Microsoft
- Product-based startups

## LEVEL 5 – Hard / Advanced Optimization

19. **Two Heaps**  
20. **Top K Elements**  
21. **K-way Merge**  
22. **Greedy Algorithms**  
23. **0/1 Knapsack (DP)**  
24. **Trie**  
25. **Bitwise XOR**  
26. **Ordered Set**  
27. **Multi-thread**

These are advanced / niche but powerful.
